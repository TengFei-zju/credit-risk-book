# 封面设计哲学：数据之盾

## 设计理念

**名称**：数据之盾 (The Data Shield)

**核心概念**：信贷风控的本质是**守护**——用数据构建防线，保护金融机构免受风险侵蚀。封面设计应传达：

1. **防护感**：盾牌形态象征风控的防御属性
2. **数据流动**：粒子流代表数据输入与风险信号
3. **科技质感**：现代、专业、可信赖的视觉语言
4. **层次分明**：从数据到决策的清晰路径

## 视觉元素

### 配色方案
- **主色调**：深海蓝 (#0A1628) —— 稳重、专业、金融属性
- **强调色**：琥珀金 (#FFB84C) —— 预警、价值、决策信号
- **辅助色**：数据青 (#00D4FF) —— 科技、流动、数据可视化
- **背景渐变**：从深邃到通透，象征从风险到安全的转化

### 构图结构
```
┌─────────────────────────────────┐
│                                 │
│     书名（居中， bold）           │
│                                 │
│    ╭─────────────────╮          │
│    │   盾形轮廓       │          │
│    │   内部数据流     │          │
│    │   粒子网络       │          │
│    ╰─────────────────╯          │
│                                 │
│   [特征工程][机器学习][图模型]   │
│                                 │
│         作者名                   │
│                                 │
└─────────────────────────────────┘
```

### 图形语言
- **盾形轮廓**：半透明、线条感，非实体填充
- **数据流粒子**：从四周向中心汇聚，象征信息收集
- **风险节点**：琥珀金高亮，表示需要关注的风险点
- **安全区域**：中心区域更亮，表示通过风控后的清晰决策

## 字体选择
- **中文标题**：思源黑体 Bold / 微软雅黑 Bold —— 现代、易读、力量感
- **英文/数字**：JetBrains Mono —— 代码、数据、技术属性
- **副标题**：思源宋体 —— 人文、专业、可信

## 情感基调
- 专业但不冷峻
- 科技感但不浮夸
- 稳重但有活力
- 复杂但清晰

## 输出规格
- 尺寸：800 x 1200 px (3:4 比例，适合书籍封面)
- 格式：PNG（高分辨率）+ PDF（打印就绪）
- DPI：300
